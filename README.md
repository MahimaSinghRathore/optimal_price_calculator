# 📊 Optimal Price Calculator

This is a simple Python GUI app to calculate the **optimal selling price** for a product to maximize profit, based on basic economic demand modeling.

---

## ✅ Features

- 📈 Calculates **quantity sold**, **revenue**, and **profit** for a range of prices.
- Finds the **optimal price** where profit is maximum.
- Displays a **Profit vs. Price** plot with the optimal point highlighted.
- Built with **Tkinter** for the GUI and **Matplotlib** for the plot.

---

## ⚙️ Technology

- **Python**
- **Tkinter**
- **Matplotlib**
- **NumPy**

---

## 🚀 How to Run

1️⃣ **Clone the repository**


git clone https://github.com/MahimaSinghRathore/optimal_price_calculator.git
cd optimal_price_calculator

2️⃣ Install dependencies
pip install matplotlib numpy
(Tkinter is usually included with Python, so no separate install is needed.)

3️⃣ Run the app
python main.py
4️⃣ Use the GUI
Enter:

Base Demand (a) — expected demand at zero price.

Price Sensitivity (b) — how demand drops with price increase.

Cost per Unit (c) — production cost per unit.

Click Calculate

View:

Optimal price & maximum profit.

Profit curve and optimal point.
